   npm install

2. Create .env file with Supabase credentials:
   EXPO_PUBLIC_SUPABASE_URL=your_url
   EXPO_PUBLIC_SUPABASE_ANON_KEY=your_key

3. Start development:
   npm run dev

4. Scan QR code with Expo Go app
